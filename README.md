# 🚀 Smart File Organizer Agent

This is a Python-based project that automatically organizes uploaded files into categorized folders (Images, Documents, Videos, Code, etc.).

## 🔧 Current Status
🚧 Project is under development

## ✨ Features (Planned / In Progress)
- File categorization based on type
- Automatic folder creation
- Clean file naming
- ZIP download of organized files
- Streamlit-based UI

## 🛠 Tech Stack
- Python
- Streamlit

## 📌 Note
This project is still being developed and will be updated soon with more features and deployment.
